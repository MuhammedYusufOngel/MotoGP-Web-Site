# MotoGP Web Sitesi  
  
## Kullandığım Teknolojiler  
- ASP.Net Core 6.0
- MSSQL
- EntityFramework
- Ajax
- SignalR
- Html
- Css
- Javascript
- Bootstrap

## Front End
- Html
- Css
- Javascript
- Bootstrap

## Back End  
- C#
- MSSQL

## Öne Çıkan Özellikler
- MotoGP ile alakalı haberler
- Şampiyona sıralaması
- Tüm yarışların sonuçları
- Özel admin paneli
- Admin için kolay yarış paneli
- Sürücüler alakalı istatistikler
- Pistlerle alakalı bilgiler

## Teknik Özellikler
- ASP.Net Core 6.0
- MSSQL Veri Tabanı
- EntityFramework
- SignalR
- Ajax
